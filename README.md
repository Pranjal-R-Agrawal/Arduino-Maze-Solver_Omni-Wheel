# Arduino-Maze-Solver_Omni-Wheels

This is the code for a maze solving robot. The robot uses the left hand rule.

Important Components -

* Atmega2560 x 1
* TB6612FNG x 2
* IR Sensor x 16
* 600 RPM N20 motor x 4
* Turnigy 460 mah 2s 20C - 40C LiPo battery x 1
